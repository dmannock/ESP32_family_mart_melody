# ESP32_family_mart_melody
Plays the Family Mart jingle via a connected speaker. Default pin is GPIO4.

## Usage
- Set ```speakerPin```
- Upload to device
- Enjoy

## ESPHome Example
- Set ```pin```
- Make sure you have these set in your ```secrets.yaml```
    - ```wifi_ssid```
    - ```wifi_password```
    - ```ota_password```
- Upload
- Enjoy
